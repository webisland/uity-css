# uity.css 0.0.14

Run UIty.css demo with a command `jekyll serve` and access via `localhost:4000`

### Constraints shortcuts
xs - small mobile\n
sm - mobile\n
md - tablet\n
lg - desktop (default)\n
xl - large desktop\n

### Properties shortcuts
\_h - horizontal\n
\_v - vertical\n
\_top - top\n
\_btm - bottom\n
